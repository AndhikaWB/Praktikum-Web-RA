Tentang Saya

Saya adalah seorang mahasiswa IT dari Institut Teknologi Sumatra (Lampung, Indonesia). Saya memiliki minat yang tinggi di bidang artificial intelligence dan embedded system. Di waktu senjang, terkadang saya berkontribusi ke beberapa software open-source seperti NewPipe, Bromite, dll. Saya adalah seorang yang perfeksionis, bertanggung-jawab, serta mampu bekerja sama dan memimpin tim.

Keahlian

  • C++ (Sedang) - Familiar dengan konsep C++ dasar (array, struct, vector, dsb) dan algoritma struktur data (linked list, stack, binary tree, dsb)
  • Python (Sedang) - Familiar dengan konsep pemrograman berorientasi objek dan game development (PyGame)
  • Web (Pemula) - Familiar dengan HTML dan CSS dasar (styling, layouting, dsb)
  • Photoshop (Pemula) - Saya umumnya menggunakan Paint.NET dan GIMP sebagai alternatif (paham konsep layer, cropping, blurring, dsb)
  • Office (Mahir) - Cukup mahir menggunakan berbagai software Office (Word, PowerPoint, Excel)
  • Embedded (Sedang) - Mampu membangun prototipe sistem tertanam sederhana (Arduino, ESP32)
